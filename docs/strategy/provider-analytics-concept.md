# Концепт: Аналитика в кабинете исполнителя

> Дата: 2026-04-26
> Статус: Концепт, ожидает спецификации

---

## Зачем

Исполнители хотят понимать как они выглядят на платформе, растут ли и где теряют.
Аналитика повышает вовлечённость и удержание — исполнитель остаётся там где видит результат.

---

## 4 блока метрик

### 1. Видимость профиля
- Просмотры профиля (день / неделя / месяц)
- Показы в поисковой выдаче
- Конверсия: показы → переходы на профиль

### 2. Бизнес-метрики
- Количество заказов (всего / завершённые / отменённые)
- Заработок за период
- Средний чек
- % завершения заказов (влияет на позицию в ранжировании)

### 3. Активность и скорость
- Среднее время отклика на заявки
- % откликов из просмотренных заявок
- Конверсия: отклик → выбрали заказчик

### 4. Репутация
- Динамика рейтинга (график изменений во времени)
- Количество отзывов за период
- % повторных клиентов

---

## Дополнительно — для тех у кого есть бизнес-витрина

- Просмотры витрины `/b/[slug]`
- Клики по контактам и соцсетям
- Просмотры каждого объявления (ServiceListing) отдельно

---

## MVP — минимальный набор (5–6 метрик)

Полная аналитика перегружает. На старте показываем только самое ценное:

| Метрика | Почему важна |
|---|---|
| Просмотры профиля | Базовый сигнал видимости |
| Кол-во и сумма завершённых заказов | Главный бизнес-показатель |
| Конверсия откликов | Помогает понять качество откликов |
| Среднее время отклика | Платформа ранжирует по нему |
| Динамика рейтинга | Мотивирует поддерживать качество |

---

## Технические заметки

- Данные считаются из существующих сущностей: Order, Proposal, Review, ProfileView (новая)
- `ProfileView` — лёгкая таблица: `providerId`, `viewedAt`, `source` (search / direct / listing)
- Агрегаты считаются на стороне сервера, кэшируются (не real-time)
- Период: 7 дней / 30 дней / всё время — переключатель

---

## Фазы реализации

**MVP:**
- Дашборд с 5 ключевыми метриками
- Период: 30 дней по умолчанию
- Простые счётчики без графиков

**Фаза 2:**
- Графики динамики (Chart.js / Recharts)
- Сравнение с предыдущим периодом (↑↓ дельта)
- Аналитика витрины (для BusinessProfile)
- Топ категорий по спросу в городе исполнителя
