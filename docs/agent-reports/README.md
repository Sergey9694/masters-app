# Agent Reports

Здесь хранятся отчёты мультиагентной системы разработки.

## Общий протокол

**Каждый агент** обязан следовать общему протоколу: `agent-protocol/SKILL.md`

Ключевые правила:
- 🛑 **Сначала думай, потом пиши код** (Pre-flight checklist)
- 🔍 **Build vs Buy** — всегда проверяй, есть ли готовая библиотека
- 💬 **Консультируйся** — новые зависимости, DB-миграции, рефакторинг > 3 файлов
- ✅ **Проверяй** — `tsc`, dev-сервер, визуал после КАЖДОГО изменения
- 📝 **Отчитывайся** — единый формат, на русском языке
