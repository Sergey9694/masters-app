## 🚀 Как проверить (пошагово)

### Сценарий 1: Проверка связи (Real-time)
1. Открой проект (`npm run dev:full`).
2. Зайди в два разных браузера (или обычный + инкогнито) под разными пользователями.
3. Начни диалог.
4. **Ожидаемый результат**: Сообщения появляются мгновенно без перезагрузки страницы.

### Сценарий 2: Переход к заказу
1. В окне чата нажми на кнопку/ссылку **"Перейти к заказу"** в верхней части.
2. **Ожидаемый результат**: Тебя должно перекинуть на страницу конкретного заказа, а не на ошибку 404.

### Сценарий 3: Безопасность
1. Попробуй вручную изменить ID чата в адресной строке на случайный набор букв.
2. **Ожидаемый результат**: Система должна выдать ошибку доступа или пустую страницу, не раскрывая данные других пользователей.
