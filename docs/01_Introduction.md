# Введение: Маркетплейс «Мастер Района»

## Концепция проекта
«Мастер Района» — это гиперлокальный маркетплейс бытовых услуг, ориентированный на жителей конкретных микрорайонов. Основная цель — создать доверенную среду для решения повседневных задач (ремонт, уборка, сантехника и т.д.) силами соседей или проверенных локальных мастеров.

### Ключевые отличия от конкурентов
1. **Гиперлокальность**: Поиск в радиусе 10-15 минут пешком. Использование PostGIS для точного гео-таргетинга.
2. **Доверие**: Проверка статуса «Местный житель» и верификация документов.
3. **Telegram native**: Работа через Telegram Web App (TWA), что дает мгновенный доступ без установки приложений и удобные уведомления через бота.
4. **Низкие комиссии**: Монетизация базируется на подписке для мастеров, а не на огромных процентах с каждого заказа.

## Бизнес-модель
- **SaaS (Подписка)**: Мастера оплачивают доступ к заказам в своем районе.
- **Верификация**: Разовая оплата за статус проверенного специалиста.
- **Реклама**: Платное поднятие в топе категорий.

## Текущее состояние
Проект находится на стадии активной разработки (Фаза 2.1). Заложен фундамент архитектуры, настроена база данных с поддержкой гео-данных, реализована авторизация через Telegram и создана форма подачи заявок.
