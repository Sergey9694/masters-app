# 🎭 Workflow: Task Delegation (Orchestrator)

Этот воркфлоу активирует роль Оркестратора для декомпозиции и делегирования сложных задач.

## Шаги

1. **Инициализация**:
   - Прочитать `.agent/skills/agent-protocol/SKILL.md`.
   - Прочитать `.agent/skills/agent-orchestrator/SKILL.md`.

2. **Анализ задачи**:
   - Оценить масштаб задачи и затронутые модули.
   - Свериться с `docs/PROJECT_STATE.md`.

3. **Декомпозиция**:
   - Разбить задачу на подзадачи (Research, Arch, Security, Code, Test).
   - Выбрать подходящих агентов для каждой подзадачи.

4. **Представление плана**:
   - Показать план пользователю и дождаться одобрения.

5. **Выполнение**:
   - Последовательно выполнять подзадачи, переключая роли (скиллы).
